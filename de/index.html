Begeben Sie sich auf ein unvergessliches Abenteuer mit dem Long Tail Boat auf Koh Samui ab nur 3 200 THB für ein komplett privates Boot mit Ihrem eigenen thailändischen Skipper. Unsere Tour führt Sie durch türkisfarbenes Wasser zu den spektakulärsten Zielen auf Koh Samui.

Ob für ein Paar in den Flitterwochen, eine Familie im Urlaub oder eine Gruppe von Freunden auf der Suche nach Abenteuer — Ihre private Long Tail Boat Tour auf Koh Samui wird der Höhepunkt Ihrer Thailand-Reise.
